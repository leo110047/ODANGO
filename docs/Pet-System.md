# 寵物系統

## 孵化流程

1. 所有用戶從「蛋」開始
2. 累積足夠 XP 後，蛋會孵化
3. 孵化時 Bot 會私訊 3 個隨機寵物選項
4. 用戶選擇後寵物會出現在桌面上

## 進化階段

| 階段 | 說明 |
|------|------|
| 🥚 egg | 初始蛋形態 |
| 🐣 stage1 | 孵化後第一階段 |
| 🐾 stage2 | 第二階段進化（部分寵物） |
| 🌟 stage3 | 最終階段進化（部分寵物） |

各寵物的進化門檻和可達階段不同。

## XP 獲取方式

| 行為 | XP |
|------|-----|
| 發送訊息 | +1 |
| 語音通話（每分鐘） | +2 |
| 遊戲事件 | +20 |

## 寵物大小

寵物大小（scale）會隨著 XP 累積而成長：
- 初始大小：1.0x
- 最大大小：1.6x

## Sprite 朝向

每隻寵物的 Sprite 有預設朝向（`defaultFacing`）：
- `left`：預設朝左
- `right`：預設朝右

動畫系統會根據移動方向自動翻轉 Sprite。

## 多寵物系統

- 用戶可以擁有多隻寵物
- 在設定視窗中可選擇要顯示哪些寵物
- 每隻寵物可個別設定移動開關和速度
- 蛋階段的寵物不會移動
